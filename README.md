# JY Prompt Library (AI 提示词专属知识库)

这个仓库专门用于沉淀和管理日常业务中高价值的 AI 提示词（Prompt），涵盖人物设定、电商视觉、场景构建、艺术风格、营销创意等核心维度。

## 📁 目录结构与分类规范

### 1. 人物角色设定 (01_Character_Design)
核心：所有关于“人/拟人”的描述模板
- 现实/写真人物、幻想/神话人物、动漫/卡通/赛博朋克风格人物
- 历史/时代人物、职业/身份角色（医生、CEO、黑客等）
- 动物拟人 / 怪物 / 机械人、年龄段专属（婴儿、儿童、老人）
- *可存储：外貌全套描述、服装&配饰模板、表情&姿势库、多角度一致性提示*

### 2. 电商场景与产品展示 (02_Ecommerce_Product)
核心：商业转化类提示词
- 单品展示（白底/生活化/悬浮）、场景化营销（生活场景、节日场景）
- 店铺首页/详情页视觉、促销/限时活动场景
- 3D渲染&产品包装、跨境电商风格（Amazon/Shein/Temu）
- *可存储：产品主图/细节图模板、生活方式场景、A/B测试版本、视频分镜提示*

### 3. 环境与场景构建 (03_Environments_Scenes)
核心：任何“空间/背景”描述
- 自然景观（森林、海洋、四季）、城市/建筑（赛博、古代、未来）
- 室内空间（卧室、咖啡店、办公室）、科幻/奇幻世界
- 历史/文化场景、天气&氛围（雨天、雾气、黄金时刻）
- *可存储：场景全景模板、镜头语言、时间段专属光影、空镜/无人物场景*

### 4. 艺术风格与视觉语言 (04_Styles_Aesthetics)
核心：决定“画风”的万能修饰词
- 摄影风格（胶片、商业摄影、电影感）、绘画风格（油画、水彩、国画、插画）
- 特定艺术家/流派、材质与质感（金属、玻璃、毛绒）
- 光影与色彩（高对比、梦幻、暗黑）、时代风格（维多利亚、80s、蒸汽波）
- *可存储：风格权重模板、跨风格融合 Prompt、负面提示词专库*

### 5. 营销与广告创意 (05_Marketing_Advertising)
核心：商业文案+视觉结合
- 品牌形象（Logo/VI/吉祥物）、广告海报/短视频
- 社交媒体内容（小红书、抖音、Instagram）、节日/节点营销
- 用户故事/UGC风格
- *可存储：Slogan+视觉一体化模板、不同平台尺寸&风格适配版*

### 6. 叙事与故事生成 (06_Narrative_Storytelling)
核心：LLM + 多模态故事提示
- 世界观构建（小说/游戏设定）、情节&分镜（漫画/短片/小说）
- 角色对话模板、情感曲线提示、IP扩展
- *可存储：System Prompt（角色扮演设定）、Few-shot 示例库、多章节连贯性提示*

### 7. 专业领域应用 (07_Professional_Domains)
核心：垂直行业专用提示
- 建筑/室内设计、时尚/服装设计、食品/餐饮视觉
- 医疗/科技/金融视觉、教育/培训素材、游戏/UI/概念设计
- *可存储：行业规范模板（真实感、比例）、技术参数专用词*

### 8. 工具参数与优化 (08_Tools_Optimization)
核心：提效类（非内容本身）
- 各大平台专属参数 (Midjourney / Flux / SD3 / Kling / Runway)
- 一致性技巧（--cref、Character Reference）、批量生成模板
- 负面词库 & 权重技巧、提示词工程方法（CoT、ToT）
- *可存储：各平台最佳实践模板、自动优化脚本提示*

### 9. 人物角色设定 (System Prompt 文本库) (09_Character_Persona)
核心：专门存放用于大模型角色扮演、虚拟助手、AI 客服的 System Prompt。
- 预设丰富的性格、习惯、背景故事。
- 提供现成可复制的系统指令（System Instructions），直接用于对话模型的角色定调。
- *包含不同职业、场景、风格的虚拟对话数字人模板。*
